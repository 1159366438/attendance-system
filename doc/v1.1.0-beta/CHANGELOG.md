# 变更日志

### 新增功能
- 用户注册功能

- 消除代码重复，统一登录和注册的验证逻辑
- 移除Service层的重复验证逻辑
- 修复TypeScript类型错误，补充缺失的常量定义

- 统一错误处理风格为ResponseResult模式
- 消除硬编码值，统一使用常量管理
