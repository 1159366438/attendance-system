# v1.1.0-beta 版本开发计划

**版本**: v1.1.0-beta  
**目标**: 用户管理（内测版）  
**状态**: 规划中  

## 版本目标

v1.1.0-beta 版本的核心目标是引入用户管理系统，包括部门管理、基于角色的访问控制（RBAC）和多表数据库设计。此版本将为多用户管理奠定基础，使系统从单用户模式转向支持多用户的企业级应用。

## 核心功能

### 1. 部门管理
- 部门的增删改查（CRUD）功能
- 部门层级结构支持（支持多级部门）
- 部门信息管理（部门名称、描述、负责人、联系方式等）

### 2. RBAC 权限系统
- 基于角色的访问控制模型
- 用户-角色-权限的关联管理
- 权限细粒度控制（功能权限、数据权限）
- 角色的创建、修改、删除功能

### 3. 数据库设计
- 多表数据库设计（用户-部门-角色-权限关联表）
- 数据库关系规范化
- 性能优化索引设计

### 4. 管理员面板
- 用户管理界面（用户信息查看、编辑、权限分配）
- 角色管理界面
- 权限分配界面

### 5. 登录认证集成
- 登录认证集成真实用户数据
- 用户会话管理
- 权限验证中间件

## 技术实现

### 后端技术
- Spring Security 集成 RBAC 权限控制
- MyBatis 实现多表关联查询
- MySQL 数据库设计与优化
- JWT 或 Session 管理用户认证

### 前端技术
- Vue 3 + TypeScript 实现管理界面
- Element Plus 组件库构建用户界面
- Axios 实现与后端 API 通信
- 权限控制路由守卫
